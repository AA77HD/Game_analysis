# Game_analysis
Descripción

Este proyecto fue realizado como parte del Bootcamp de Ciencia de Datos de TripleTen.
El objetivo fue analizar datos de ventas y calificaciones de videojuegos para entender qué factores influyen en su éxito, practicar análisis exploratorio, visualizaciones y pruebas de hipótesis con Python.

Öbjetivos

Identificar las plataformas y géneros con más ventas.
Analizar las tendencias de lanzamientos y ventas a lo largo de los años.
Explorar cómo las calificaciones (de críticos y usuarios) afectan las ventas.
Comparar las preferencias de los jugadores por región.

Dataset

Contiene información sobre más de 16.000 videojuegos:
Plataforma (PS, Xbox, Wii, etc.)
Género
Ventas por región (NA, EU, JP, Other)
Puntajes de críticos y usuarios
Clasificación ESRB (edad recomendada)

Análisis y Resultados

El mayor número de lanzamientos ocurrió entre 2007 y 2011, con PS3, Wii y Xbox 360 dominando el mercado.
Los géneros más vendidos fueron Acción, Deportes y Shooter.
Se encontró una correlación moderada (0.4) entre las reseñas de críticos y las ventas, pero casi nula con las de usuarios.
En Japón destacan los juegos de Rol (RPG) y consolas Nintendo, mientras que en Norteamérica y Europa predominan Acción y Deportes.

Conclusiones

Las consolas tienen un ciclo de vida aproximado de 8 a 10 años.
El PS2 fue la consola más vendida históricamente.
Las estrategias de marketing deben adaptarse según la región.
Las buenas reseñas de críticos pueden impulsar significativamente las ventas.

Herramientas

Python • Pandas • NumPy • Matplotlib • Seaborn • SciPy • Jupyter Notebook
